Повинен бути встановлений python версії 3.8.
Наступні кроки залежать від операційної системи.
Команди викликаються з кореневої папки проекту.

Unix-like OS:
sh setup.sh
export FLASK_APP=run.py
./venv/bin/python -m flask run

Windows OS:
setup.bat
set FLASK_APP=run.py
flask run


Після вказаних дій необхідно зайти на http://127.0.0.1:5000/ в браузері.
